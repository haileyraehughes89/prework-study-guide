HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three different types of style sheets:
      • Inline CSS:
  • Useful for quick changes
  • Tedious to make changes
      • Internal CSS:
  • Embeds an entire style sheet directly into HTML
  • Will style multiple elements in page code
      • External CSS:
  • most preferred due to easy editing

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Functions are a set of instructions for the computer to read
  • To define a function:
      declare the function
      name the function
      write function instructions in the code block
  • Call the function by naming it followed by ();
  • To avoid errors, code according to control flow
